Recidivism Forecasting

A Random Forest Classifier trained using recidivism data to output the probability of recidivism, a value between 0 and 1, based on inmate data related to demographics, security, and behaviors. Uses and compares accuracy and AUC of probability of recidivism between Random Forest, SLIM, and Neural Network models. Uses numpy, matplotlib, tensorflow, keras, and sklearn libraries.

Code is in Python via Google Colab.
